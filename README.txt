This svn directory contains the source for the Apache Karaf web site visible at http://karaf.apache.org/

You can preview your changes using
mvn jetty:run -Plive

More information is available at
   http://localhost:8080/index/community/faq/how-do-i-edit-the-website.html

or inside this folder at the following location:
   src/main/webapp/index/community/faq/how-do-i-edit-the-website.page
